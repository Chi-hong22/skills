# Research Weekly Report

## 核心规则（必读）

| 规则 | 说明 |
|------|------|
| **Tags** | 保留 `日志/周报`；过滤其他 `日志/` 前缀；提取非日志类 tags |
| **时间格式** | `YYYY-MM-Do HH:mm:ss dddd`（如 `2026-01-22nd 10:36:16 Thursday`） |
| **图片处理** | 正文用"见图X"引用，附图索引插入 `![[image.png]]` |
| **公式格式** | 行内 `$...$`，独立 `$$...$$` |
| **来源追踪** | 关键信息后标注【日期·分类】 |
| **语气风格** | 学术报告式 + 克制反思（"进一步意识到..."） |
| **板块选择** | 只含有实际内容的板块，禁止"待补充" |

## 工作流程

1. **读取日报** → 提取 tags、日期范围、按项目聚类
2. **读取图片** → 理解内容，生成准确描述
3. **填充板块** → 按项目组织，灵活选择9大板块
4. **格式化** → LaTeX 公式、来源标注、图片索引
5. **输出** → 保存到 `/04_自我管理/00_日志/{YYYY}/{MM}/周报_{YYMMDD}.md`

## 输出格式

```yaml
---
标题: 周报_YYMMDD
tags:
  - 日志/周报
  - [从日报提取的非日志类 tags]
创建时间: YYYY-MM-Do HH:mm:ss dddd
编辑时间: YYYY-MM-Do HH:mm:ss dddd
---
```
