# Research Weekly Report - References

本文件夹包含科研周报生成的参考文档。

## 文件说明

### prompt.md
完整的科研周报生成提示词，包含：
- Background: 目标、数据形态、核心需求
- Profile: 角色定位、能力描述、风格要求
- Skills: 逻辑聚类、全要素映射、图片索引、来源追踪、公式格式化
- Goals: 生成目标和9大核心板块定义
- Constraints: 约束条件和格式要求
- Workflow: 详细的处理流程
- OutputFormat: 输出格式规范和示例
- Example: 输入输出示例

**使用场景**: 需要深入理解周报生成逻辑、9大板块映射规则、公式格式化要求时阅读。

### example_output.md
实际的成品周报示例（周报_260118），包含：
- 完整的 frontmatter（标题、tags、时间戳）
- 3个研究项目的完整周报内容
- 9大核心板块的实际填充示例
- 图片索引和引用示例
- 来源追踪标注示例【日期·分类】

**使用场景**: 作为生成周报时的参考模板，了解最终输出的格式和质量标准。

### obsidian_template.md
Obsidian 模板插件的周报模板文件，包含：
- Templater 脚本（自动生成标题、日期、路径）
- Frontmatter 元数据定义
- 文件自动处理函数（创建文件夹、移动文件、处理重名）

**使用场景**: 理解 Obsidian 中的元数据设置和文件管理逻辑，了解自动化路径计算规则。
